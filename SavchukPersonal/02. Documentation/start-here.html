Getting started

If you already have experience in web design and template editing, uploading, and server environment then you can skip this section.

Requirements

Excluding the language skills mentioned before you need also proper software to edit the template files.

You can find a recommendation for software in the support section at the bottom of the page.

You also need a hosting service where you can upload your files to the internet.


Installation

This is a very generic guide for installing/uploading an HTML template.

If your hosting provider is providing a guide on how to upload a static HTML website/template you should use it instead.

  1. Get your files

    Open the HTML files folder that is including your main template files.

  2. Log In

    Log into your hosting space via FTP software ( Filezilla or Cyberduck for example ).

  3. Upload

    Upload all files from your HTML files folder into another folder on your hosting space via FTP software.


Local Server Environment

You have to set up a Local Server Environment to use the image hover effect on your local device due to Javascript AJAX.

Chrome Web Server - Local Server Environment in your chrome browser ( Very easy to set up )

MAMP - Local Server Environment for MAC & Windows

Essentials

Background

Choose from 6 different and unique background effects.
You can change and modify the background via the included config.js. file for mobile and non-mobile devices under section 4. BACKGROUND CONFIG.


var option_hero_background_mode = "image";

var option_hero_background_mode_mobile = "image";


Image Hover Effect

Notice!

The image hover effect requires a local server environment to function due to Javascript AJAX.

All settings including the image selection and effect can be modified via the included config.js. under section 3. PROFILE PICTURE CONFIG.


Additionally, you can create your own hover effect with the guide below.

  1. Set to Custom

    Set the value from var config_profile_image_effect = "tech"; to "custom" in your config.jsfile.

  2. Create Image

    Create or save an image in your assets/images/effect_maps folder and name it custom.jpg.



Color Scheme

You can change the color scheme via the variable var config_color_scheme = "warning_yellow"; in your config.js. file under section 2. BASIC CONFIG.



Scrollbar

You can select from 3 different scroll indicators via the variable var config_scroll_bar = "bar"; in the included config.js. file under section 2. BASIC CONFIG.



Cursor

You can select from 3 different cursor setups via the variable var config_cursor_mode = "cursor_2"; in the included config.js. file under section 2. BASIC CONFIG.



Contact Form

Notice!

PHP mail() function support is required for contact form function.

Find out how to set up the contact form below.

  1. Open File

    Open the assets/php/contact.php file.

  2. Change Email

    Locate the following line of code in your contact.php file and replace your@mail.com with your email address.


    $emailTo = "your@mail.com";
  3. Save

    Don't forget to save the contact.php file.



Scrolling Animations

Add scroll animations to any element inside the <div class="section-container"></div> DOM element.

You only have to add the class scroll-animation to an element to activate the default scroll animation. You can modify the default animation in your config.js file.


Additionally, you can configure each element independently by adding data attributes as seen below.


                    
  <div class="scroll-animation" data-animation="rotate_up" data-animation-duration="1.5" data-animation-ease="power4.out" ></div>

Data scroll animation settings:


Option Values Info
data-animation=
  • "fade_from_left"
  • "fade_from_right"
  • "fade_from_bottom"
  • "rotate_up"
 * Animation of the element
data-animation-duration=
  • "1.2"
 * Animation Duration of the element.
The value number equals seconds. Set any number
data-animation-ease=
  • "power1.out"
  • "power2.out"
  • "power3.out"
  • "power4.out"
  • "expo.out"
  • "sine.out"
  • "none"
 * Ease of the Animation
